## 📖 About

Waste2Grow is a smart waste management platform designed to minimize organic waste by connecting waste generators with collectors and farmers. The platform encourages sustainable recycling practices, supports compost production, and promotes environmental awareness through a modern and intuitive web application.

---

## ✨ Features

- 🌱 Organic waste management
- 🤖 AI-powered waste classification
- 📍 Location-based waste collection
- 🚚 Pickup scheduling
- 👨‍🌾 Farmer dashboard
- 📊 Analytics and insights
- 🔒 Secure authentication
- 📱 Fully responsive design
- ⚡ Fast and optimized performance

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/waste2grow-frontend.git
```

### Navigate to Project

```bash
cd waste2grow-frontend
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

Open:

```
http://localhost:5173
```
